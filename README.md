# 🧠 Medical Prediction using Gradient Boosting

## 📌 Project Overview

This project uses Machine Learning to predict:

* Breast Cancer (Malignant vs Benign)
* Heart Disease (Presence vs Absence)

## ⚙️ Algorithms Used

* Gradient Boosting Classifier
* Hyperparameter Tuning (RandomizedSearchCV)

## 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

## 📈 Results

* Breast Cancer Model achieved very high performance
* Heart Disease model showed balanced results
* Hyperparameter tuning improved both models

## 📁 Dataset

* Breast Cancer Dataset
* UCI Heart Disease Dataset

## 🚀 How to Run

```bash
pip install -r requirements.txt
python train.py
```

## 📌 Key Insights

* Gradient Boosting performs excellently on tabular medical data
* Recall is critical for disease detection
* Feature importance helps interpret model decisions
